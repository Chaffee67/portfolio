Ma Rando

Landing page - Application mobile - Design d'interface - 2022/2023
"Trouver votre parcours idéal pour découvrir de nouveaux horizons !"

Le client, la FFR, souhaite communiquer sur sa nouvelle application via le site marando.fr, site one page de l’application. Le site a pour but de présenter l’application et ses fonctionnalités, inciter au téléchargement de l’application. Il souhaite également y intégrer un jeu concours, en bas de page, pour construire une base de données de leads. La demande est de créer une page d’accueil pour expliquer le but de l’application et de modifier l’interface de celle-ci avec ses différentes écrans (accueil, recherche de circuits, randonnée, etc.).
